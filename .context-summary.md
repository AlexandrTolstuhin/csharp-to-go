- Блок-схема выбора технологии
- Паттерны: идемпотентность (INSERT ON CONFLICT), graceful shutdown (signal+context+WaitGroup), retry с exponential backoff и jitter, Dead Letter Queue, сериализация (JSON/Protobuf/Avro/MessagePack), Outbox Pattern (dual write проблема, outbox table + relay), Saga Pattern (хореография vs оркестрация)
